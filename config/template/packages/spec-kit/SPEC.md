# Product SPEC — Fitted
## Primary Flow (canonical 8)
1. Welcome — “Meet Your Digital Twin” → CTA Get Started
2. How It Works — Capture (2 photos), Create (3D twin), Shop (size recs) → Continue
3. Permissions — camera + privacy copy → Allow Camera Access
4. Setup — live preview, outline, “Step back 6–8 ft” → CTA enabled when positioned
5. Front Capture — 10s countdown, auto-capture, LiDAR flag
6. Rotate / Side Capture — “Turn 90° left”, 5s, auto-capture
7. Processing — ~30s stages: Analyzing → Creating 3D model → Calculating measurements
8. Results / Avatar — Height, Chest, Waist, Hips; “View All Measurements” / “Start Shopping”
## Timers
- Front 10s, Rotate 5s, Side 5s. Success → Processing → Results.
## LiDAR
- Banner “Active depth sensing” when available; explicit fallback notice if not.
## Commerce Handoff
- “Your Size” badges injected into product listing; web storefront primary.
